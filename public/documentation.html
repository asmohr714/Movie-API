Movie API

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Supported Requests

Request URL HTTP Method Request body data format Response body data format
Get list of movies /movies GET None An array of JSON objects holding data about all the movies
Get list of movies by director /movies/directors/[director name] GET None An array of JSON objects holding data about all the movies by requested director
Get list of movies by genre /movies/genres/[genre] GET None An array of JSON objects holding data about all the movies by requested genre
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about the requested movie
Add a movie /movies POST A JSON object holding data about the movie to add
Remove a movie by ID /movies/id/[id] DELETE None A text message indicating whether the movie was successfully removed
Add a user /users POST A JSON object holding data about the user to add A JSON object holding data about the user that was added
Remove a movie from a user's favorites list /users/[name]/[movie] DELETE None A text message indicating whether the movie was successfully removed from the user's favorited movies
